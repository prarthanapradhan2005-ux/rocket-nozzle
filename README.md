# Rocket Nozzle Design Study

## Objective
To study the variation of Mach number with area ratio in convergent and convergent-divergent rocket nozzles.

## Theory
Compressible flow relations were used to determine nozzle flow characteristics.

## Governing Equation
Area-Mach relation for isentropic flow.

## Tools Used
- Python
- NumPy
- Matplotlib
- VS Code

## Results
Generated Area Ratio vs Mach Number graph.
